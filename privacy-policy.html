隐私政策

河马 WhatsApp 翻译助手 Chrome 扩展程序

最后更新日期：2025年5月13日

我们非常重视您的隐私。本插件不注册账号、不收集个人信息、不追踪用户行为。所有设置数据仅保存在您的本地浏览器中。

1适用范围

本隐私政策适用于「河马 WhatsApp 翻译助手」Chrome 浏览器扩展程序（以下简称"本插件"）。本插件是一款第三方工具，与 WhatsApp LLC 及其母公司 Meta 无任何官方关联。

2我们收集的信息

本插件不收集、不存储、不上传任何可识别您身份的个人信息，包括但不限于：

  • 姓名、邮箱、手机号等个人资料
  • WhatsApp 账号信息或联系人列表
  • 聊天记录、消息内容（本插件仅在您主动点击「译」按钮时读取单条消息文本用于翻译）
  • 设备标识符、IP 地址、位置信息
  • 浏览历史或行为数据

3本地存储的数据

插件使用 Chrome 的 storage.sync 接口在本地保存您的偏好设置，包括：

数据项用途存储位置
翻译方式选择记住您上次选择的翻译服务本地浏览器
目标语言记住您设置的翻译目标语言本地浏览器
是否自动翻译记住开关状态本地浏览器
API Key（可选）用于调用您选择的 AI 翻译服务本地浏览器，加密存储

以上数据仅保存在您的浏览器本地，我们无法访问，也不会同步到任何服务器。

4消息内容的处理方式

当您点击某条消息旁的「译」按钮时，该条消息的文本内容会被发送至您所选择的翻译服务进行翻译。具体如下：

翻译方式数据发送至隐私说明
谷歌翻译（免费） Google 翻译接口 适用 Google 隐私政策
OpenAI（GPT） OpenAI API 服务器 适用 OpenAI 隐私政策
Anthropic（Claude） Anthropic API 服务器 适用 Anthropic 隐私政策
DeepSeek DeepSeek API 服务器 适用 DeepSeek 隐私政策
自定义 API 您填写的自定义地址 由您自行承担风险
⚠️ 请注意：翻译敏感或私密内容时，请了解对应翻译服务的数据使用政策，本插件对第三方服务的数据处理方式不承担责任。

5我们不会做的事

  • 不在后台读取您的 WhatsApp 消息
  • 不将任何数据发送至我们自己的服务器
  • 不出售、共享或交换您的任何数据
  • 不在您未主动操作时访问消息内容
  • 不使用 Cookie 或任何跨站追踪技术
  • 不访问您的摄像头、麦克风或文件系统

6所需权限说明

本插件在 manifest.json 中声明了以下权限，以下说明每项权限的必要性：

权限用途
storage在本地保存您的翻译设置和 API Key
tabs向当前 WhatsApp 标签页发送翻译指令（弹窗点击「翻译当前消息」时使用）
host: web.whatsapp.com在 WhatsApp Web 页面注入翻译按钮
host: translate.googleapis.com调用谷歌免费翻译接口
host: api.openai.com 等调用您选择的 AI 翻译服务接口

7儿童隐私

本插件不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

8隐私政策的变更

如本隐私政策发生重大变更，我们将通过更新本页面的「最后更新日期」进行通知。建议您定期查阅本页面。
